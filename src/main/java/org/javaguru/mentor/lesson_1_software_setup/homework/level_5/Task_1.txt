Создаём первый класс!

Чтобы убедиться, что теперь вы можете писать программы на Java,
создайте класс "HelloWorld".

Для этого создайте свой пакет в src/main/java/org/javaguru
и назовите его student_yourname_yoursurname.

Внутри этого пакета создайте пакет lesson_1.

Внутри пакета student_yourname_yoursurname/lesson_1
создайте класс HelloWorld

Внутри класса создайте "main" метод.

    public static void main(String[] args) {

    }

Внутри метода "main" напишите: System.out.println("Hello World!");

    public static void main(String[] args) {
        System.out.println("Hello World!");
    }

Нажмите правой кнопкой мыши в любом месте в созданном классе и выберите Run.

Если все установлено корректно, то в консоли вы увидите текст "Hello World!"
